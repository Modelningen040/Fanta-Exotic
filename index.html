Arpanet

Arpanet var en av de första formen av ett slags internet som vi fick se, den skapades under kalla kriget. Det var ett paketväxlande nätverk. En av de första mailen skickades via Arpanet mellan 2 st datorer som sstod sida vid sida, 1971 var det när detta hände och 1973 bestod arpanet trafiken av 75%. Arpanet var skapad av Darpa under 1960 - talet.


FTP

FTP står för File Trasfer Protocol, ftp är en av de första filöverförningsprotokollen för internet. Med hjälp av ftp kan man överföra komandobaserade prtokoll för överföring av text och binära datafiler. 1971 så skrevs den första specfikationen. FTP är och ett rätt så osäkert protokoll, det är pågrund av att lösenord och data skickas i klar textformat vilket ger den en risk att någon kan sno information från dataströmmen.


Telnet

Telnet är ett textbaserad nätverksprotokoll för internet, den är textbaserad på olika men också för en annan textbaserd kommunikation. Den kan till e.x vara textbaserad mellan automatiserade procosser. Protokollet definerar avanserade fuktioner, men även kan den användas till enkla textbaserad kommunikation.

HTML

HTML, står för HyperText Markup Language, HTML är ett codings sidospråk, ett verktyg man använder för att editera de olika delarna på en hemsida så som rubrik, brödtext, backgrounds färgen på hemsida osv. Det finns olika versioner av HTML, den senaste versionen av HTML är 5.1. HTML 4.01 finns i tre olika former, det finns en som heter Frameset, strict och Transitionoal.


HTTP

HTTP, står för Hypertext Transfer Protocol, HTTP är det kommmunikationsprotokollet som överför webbsider på WWWW, World Wide Web på Internet. Den riktiga användningen för HTTP var att tillhandahålla metoden för att kunna överföra HTML hemsidor webbservrar till webbsidor.
